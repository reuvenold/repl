installation tutorial




step 1: download

go to the installation website and
click the TicTacToe.unix button next
to the linux and apple icons


step 2: open bash

go to the folder
that contains TicTacToe.unix,
right click the screen, and choose
open in terminal



step 3: setup

in the bash terminal
that appears, type:

~\Downloads
$ chmod u+x ./TicTacToe.unix



step 4: start the app

do this step every time you want to start the app

if the bash terminal is not
opened yet, repeat step 2.
when the terminal is open,
type the following line:

~\Downloads
$ ./TicTacToe.unix
